File class is used to get information about files and directories
I/O Streams are more flexible -> Work with text and binary data (like images, audio, PDFs).

Types of Streams

1)Byte Streams->Work with raw binary data (like images, audio, and PDF files).
Examples: FileInputStream, FileOutputStream.

2)Character Streams->Work with text (characters and strings). These streams automatically handle character encoding.
Examples: FileReader, FileWriter, BufferedReader, BufferedWriter.

Use character streams when working with text, and byte streams when working with binary data.
